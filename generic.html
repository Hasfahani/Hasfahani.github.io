About the Game

Arcade space duels vs a reactive CPU — simple to learn, intense to master.

What is it?

Computer Player Space Shooter is a tight, local arcade dogfight built with Python & Pygame. You pilot the yellow ship; the CPU commands the red. Choose a difficulty (Easy/Medium/Hard), dodge incoming fire, snag health power-ups, and drop your opponent to zero. The focus is responsiveness, readable visuals, and quick restarts.

AI behavior scales with difficulty: movement cadence and fire chance increase as you climb from Easy to Hard. Bullets despawn at the edges, the HUD tracks health/score, and a bold win screen invites instant replay.

Features at a glance

  • Solo vs. CPU with difficulty-scaled movement & firing
  • Three modes: Easy (chill), Medium (balanced), Hard (sweaty)
  • Health & Score HUD; crisp hit feedback & SFX
  • Power-ups (+1 health) that swing momentum
  • Pause/Resume (Esc), instant replay after a win
  • 60 FPS feel in a clean, readable arena

Install & Run

  1. Install Python 3.10+ and Pygame: python -m pip install pygame
  2. Place assets next to main.py: spaceship_yellow.png, spaceship_red.png, space.png, main-menu-pic.png, Gun+Silencer.mp3, Grenade+1.mp3, before-the-countdown.mp3
  3. Run the game: python main.py

Tip: Avoid spaces in file names (e.g., main menu pic.png → main-menu-pic.png) and update the paths in your code.

FAQ

Can I remap keys?
Not yet. Defaults: Move W/A/S/D, Fire Enter, Pause Esc.

Controllers?
Keyboard only for now.

Two-player local?
On the roadmap (shared keyboard or controller support).

Performance?
Targets 60 FPS on typical laptops/desktops.
